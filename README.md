##Laravel Educational Platform

A feature-rich Laravel backend for an educational platform. This application provides a seamless learning experience with both free and paid courses, integrated with a robust testing and quiz system to assess student progress.


##Features:
Dual Course System: Offers both free and premium (paid) courses.
Exam & Quiz Engine: Integrated system for creating and taking exams.
Laravel Backend: Built with the robust and modern Laravel PHP framework for security and scalability.
Student Dashboard: A dedicated area for students to manage their courses and track progress.
